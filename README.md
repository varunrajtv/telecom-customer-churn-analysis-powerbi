# Telecom Customer Churn Analysis Dashboard (Power BI)

## Overview
This project is a Power BI dashboard built to analyze customer churn in the telecom industry. It focuses on identifying churn patterns, customer behavior, and the key factors influencing customer loss.

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Visualization

## Key KPIs
- Total Customers
- Churned Customers
- Churn Rate %
- Total Revenue
- Average Monthly Charge

## Dashboard Insights
- Churn by Contract Type
- Churn by Tenure Group
- Churn by Internet Type
- Churn by Payment Method
- Churn by Premium Tech Support
- Churn by Churn Category

## Key Business Findings
- Customers with month-to-month contracts show higher churn.
- Customers in early tenure ranges contribute significantly to churn.
- Internet service type and payment method influence churn behavior.
- Customers without premium tech support are more likely to churn.
- Churn categories reveal major reasons behind customer loss.
